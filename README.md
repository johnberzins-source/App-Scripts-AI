# App-Scripts-AI
A lightweight chatbot built with Google Apps Script that:
- extracts topics from user messages
- pulls information from Wikipedia and related sources
- generates responses using a Markov chain
- adapts tone based on domain (tech, food, places, people)

## Features
- No external libraries
- No paid APIs
- Runs entirely on Google Apps Script
- Extensible domain system

## Limitations
- Not a true LLM
- Markov-based generation
- Depends on public encyclopedic sources
